# Masterizer
#### MZ-909 Masterizer

## Features

The MZ-909 Masterizer features a 16-channel polyphonic input, Hi-Fidelity output and active on/on switch. Offering sub-millisecond latency we believe it is the best in class available for VCV Rack.
